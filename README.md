# Flagman Xeno v5.0

**Полнофункциональный скрипт для Roblox с поддержкой Xeno**

![Version](https://img.shields.io/badge/version-5.0-red)
![Xeno](https://img.shields.io/badge/Xeno-Compatible-brightgreen)

## 📌 Особенности
- Неоновое меню с эффектом стекла
- Поиск функций в реальном времени
- 18+ функций (Fly, Noclip, Godmode, ESP, Aimbot и др.)
- Бинды на любые клавиши (ПКМ по кнопке)
- Единственный хоткей: `X` — Spider
- Полная совместимость с Xeno

## 🚀 Установка
1. Скачайте `Flagman_Xeno_v5.0.lua`
2. Откройте Xeno и подключитесь к игре
3. Вставьте код в окно выполнения
4. Нажмите **Execute**
5. Нажмите **INSERT** для открытия меню

## ⌨️ Управление
| Клавиша | Действие |
|---------|----------|
| `Insert` | Открыть/закрыть меню |
| `X` | Включить/выключить Spider |
| `ПКМ` по кнопке | Привязать функцию к любой клавише |

## 📋 Функции
- Fly, Noclip, Godmode, Spider, Scaffold
- ESP, Aimbot
- Speed x2/x3, Jump x2/x3
- Clear Parts, Kill All, Teleport

## 🎮 Бинды
1. Откройте меню (`INSERT`)
2. Нажмите **ПКМ** по любой кнопке
3. Нажмите любую клавишу
4. Готово!
