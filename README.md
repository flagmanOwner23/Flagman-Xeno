# Flagman Xeno v5.2

**Новый полёт: зависание + WASD + Space/Shift**

![Version](https://img.shields.io/badge/version-5.2-red)
![Xeno](https://img.shields.io/badge/Xeno-Compatible-brightgreen)

## 🚀 Новый полёт
- **Зависание** в воздухе (не падаете)
- **WASD** — движение вперёд/назад/вправо/влево
- **Space** — подъём вверх
- **Shift** — спуск вниз
- **Скорость** регулируется (по умолчанию 50)

## 📌 Остальные функции
- Noclip, Godmode, Spider [X], Scaffold
- ESP, Aimbot (ПКМ)
- Speed x2/x3, Jump x2/x3
- Clear Parts, Kill All, Teleport

## ⌨️ Управление
| Клавиша | Действие |
|---------|----------|
| `Insert` | Открыть/закрыть меню |
| `X` | Spider (вкл/выкл) |
| `ПКМ` по кнопке → скролл | Бинд на колёсико |

## 🎮 Бинды на колёсико
1. Откройте меню (`INSERT`)
2. Нажмите **ПКМ** по любой кнопке
3. **Прокрутите колёсико** (вверх или вниз)
4. Готово!
